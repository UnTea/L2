Что выведет программа? Объяснить вывод программы.

```go
package main

import (
	"fmt"
)

func main() {
	a := [5]int{76, 77, 78, 79, 80}
	var b []int = a[1:4]
	fmt.Println(b)
}
```

Ответ:  
Вывод: [77, 78, 79]

Создали слайс `b` от массива `a` от индекса 1 включительно до индекса 4 не включая его, который ссылается на один и тот же массив.
